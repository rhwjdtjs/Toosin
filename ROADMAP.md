### ✅ MVP-3: 학습하는 AI & 기본 전투 완성 (AI & Combat Completion)
>
> **핵심**: "플레이어를 학습하고 상대하는 '똑똑한 적'을 구현한다."

- **기간**: 2026.02.11 ~ 2026.02.15 (완료됨)

#### ⚙️ 공용 전투 시스템 정비 (AI & 플레이어 공통) - `완료`

- [x] **스탯 시스템 확장**: `FTSCharacterStats`에 방어력(Defense), 기본 공격력(BaseAttackPower) 추가.
- [x] **데미지 공식 구현**: 무기별 데미지 + 공격유형별 배율(경공격 1.0x / 강공격 1.8x / 콤보 보너스).
- [x] **피격 몽타주 시스템**: 캐릭터별로 다른 피격 몽타주 적용.
- [x] **가드 판정 세분화**: 무기-무기 충돌(가드 성공, 가해자 무기 튕김) vs 몸 피격(가드 실패, 일반 피격 처리).
- [x] **가드 에임 오프셋**: 가드 시 공격 방향에 따라 무기 위치 조절.
- [x] **라운드 기반 학습 환경**: `ATSArenaGameMode` 구현 (라운드 시작/종료/리셋), 아레나 맵 구축.

#### 🧠 AI 기초 (AI Foundation) - `완료`

- [x] **AIController & Behavior Tree**: `ATSEnemyController` 생성, 블랙보드 설정, 라운드 시작 시 BT 자동 실행.
- [x] **거리 기반 포지셔닝**: 최적 공격 거리 유지 (전진/후퇴/측면 이동).
- [x] **기본 공격 로직**: 경공격/강공격 랜덤 선택 + 쿨타임 관리.
- [x] **기본 방어 트리거**: 플레이어 공격 감지 시 가드 진입/해제.

#### 🤖 적응형 AI (Adaptive AI) - `완료`

- [x] **패턴 학습**: 플레이어의 공격 빈도 분석 (1타 위주 vs 콤보 위주), 가드/패링 성공률 분석.
- [x] **대응 로직**: 학습된 데이터에 따라 가드/공격 확률 및 반응속도 동적 조정.
